# Todo-MERN

That obligatory Todo app with an extra feature (service workers)

### Spec

1. Add Todo
    1. Set reminder (time)
2. Remove todo
3. List todo
    1. Active items
    2. Completed items
4. Mark complete
    1. Remaining count
